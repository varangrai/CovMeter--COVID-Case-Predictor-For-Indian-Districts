# CovMeter: COVID Case Predictor For Indian Districts
 Time Series Predictions conducted on active, death and recovery cases on a district level dataset.  
 
 Models used:  
    1) LSTM  
    2) Peephole RNN  
    3) Peephole GRU  
    4) Highway LSTM  
    5) Highway Encoder Decoder  
    6) Encoder Decoder  
    7) Attention Encoder Decoder  
  
 Frameworks Used:  
    1) PyTorch  
    2) Keras  
    3) Tensorflow
